Streamlit

Streamlit is a versatile platform designed to build, deploy, and share powerful generative AI applications. It provides a wide range of libraries within the AI ecosystem, enabling developers to create LLM-powered applications swiftly and efficiently.

Streamlit offers numerous examples of prebuilt apps on custom LLMs, showcasing its capabilities in performing various tasks. Below are some examples of functions that can be built using Streamlit:

LLM Examples with Streamlit

Streamlit supports any LLM with a Python implementation or API call, including popular open-source and closed-source models like OpenAI, LLaMa, Anthropic, PaLM, and models hosted on HuggingFace or built with Transformers.

  1. Build a Chatbot with Custom Data Sources Powered by LlamaIndex

    1. Augment LLMs with Data Using RAG (Retrieval Augmented Generation) Pipelines

      This process consists of two main stages:

      • Indexing Stage: LlamaIndex prepares the knowledge base by ingesting data and converting it into Documents. It parses metadata from these documents (text, relationships, etc.) into nodes and creates queryable indices from these chunks into the Knowledge Base.

      • Querying Stage: Relevant context is retrieved from the knowledge base to assist the model in responding to queries. This stage ensures the model can access data not included in its original training data.

https://blog.streamlit.io/build-a-chatbot-with-custom-data-sources-powered-by-llamaindex/

  1. Build a Chatbot Using LangChain + Streamlit

    LangChain is a framework that supports building AI agents for multiple functions. Below are some LangChain-supported tasks that can be integrated with Streamlit:

    • StreamlitChatMessageHistory: Helps LLMChain remember messages from conversations. Messages are stored in Session State across re-runs automatically.

    • LangChain: Chat with Documents: Create a chatbot capable of answering queries by referring to custom documents.

    • LangChain: Chat with SQL DB: Build a chatbot to communicate with your database.

    • LangChain: Chat with Search: Develop a chatbot that can perform search operations.

    • LangChain: Simple Feedback: A chat app that allows users to provide feedback on responses using streamlit-feedback and link to the traces in LangSmith.

https://github.com/langchain-ai/streamlit-agent

NLUX (for Natural Language User Experience)

NLUX (Natural Language User Experience) is an open-source JavaScript and React JS library designed to integrate powerful large language models (LLMs) like ChatGPT into web applications with minimal code. This library aims to provide high-quality conversational AI interfaces that are easy and quick to use.

Key Features:

  • Rapid AI Chat Interface Creation: Build conversational AI interfaces in minutes.

  • React Components & Hooks: Includes <AiChat /> for UI and useChatAdapter for integration.

  • LLM Adapters: Supports integration with ChatGPT, LangChain, LangServe APIs, Hugging Face, and more.

  • Customizable Interface: Allows for extensive UI customization through theming and layout options.

  • Zero Dependencies: Lightweight with no external UI libraries required.

  • Streaming Output: Supports streaming responses to the UI generated by the LLM.

Design Principles:

  • Intuitive: Focuses on natural and obstacle-free user interactions.

  • Performance-Oriented: Optimized for fast loading, rendering, and response times.

  • Accessible: Designed to be usable by people with disabilities across various devices and environments.

  • Developer Experience (DX): Prioritizes ease of use, understanding, and extension for developers with comprehensive documentation and support.

https://github.com/nlkitai/nlux?tab=readme-ov-file
