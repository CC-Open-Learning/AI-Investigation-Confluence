AI Investigation : Microsoft Copilot Limitations

Created by Josslyne Shantz (Unlicensed) on May 30, 2024

There are some limitations to using Microsoft Copilot. I have included the ones I could find here.

Quota

There is a maximum amount of messages that can be sent by users or integrations in a minute to a single Copilot. The quota is 800 messages a minute per Dataverse environment. If a user sends a message after the quota has been reached, they receive a notice telling them that the message failed to send and a prompt to retry.

Omnichannel message size limit

When using Copilot Studio in Microsoft’s Omnichannel, messages are limited to being at most 28 kb in size.
