AI Investigation : Prompt Engineering

Created by Jongeon, last modified on Feb 18, 2025

Prompt engineering is the practice of designing and refining prompts to optimize AI-generated responses. It involves structuring queries effectively, specifying desired formats, and providing clear instructions to achieve more accurate, relevant, and high-quality outputs.

1. Basic Principles of Prompting

  • Assigning Roles and Personas

    • Giving the model a specific role helps generate more consistent and natural responses.

    • Example: "You are a data science expert. Explain data analysis techniques using Python."

  • Controlling Response Format

    • Requesting responses in a specific format (e.g., JSON, lists, tables) increases the chances of getting the desired output.

    • Example: "Convert this information into JSON format."

  • Formatting Affects Output

    • Adjusting spacing, sentence structure, and formatting can influence the quality of the response.

2. How to Write Effective Prompts

  • Transforming Information into Knowledge

    • Instead of merely requesting information, design prompts to extract meaningful insights.

  • Use Positive Instructions Instead of Negative Ones

    • ❌ "Do not use this word." → ✅ "Include this word in your explanation."

  • Use Lists and Line Breaks Instead of Continuous Text

    • Writing in a list format improves readability.

3. Effective Prompting Strategies

  • Using Domain-Specific and Topic-Based Prompts

    • Predefining prompts tailored to specific topics enhances efficiency.

    • Example: "Prompt for generating marketing campaign slogans," "Prompt for data analysis questions"

  • Reinforcing Important Keywords

    1. Repeating key terms ensures better AI comprehension.

    2. Example:
