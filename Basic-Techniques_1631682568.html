AI Investigation : Basic Techniques

Created by Jongeon on Feb 18, 2025

1. Clarity

  • Writing prompts with specificity improves the quality of the model's response.

  • ❌ Unclear Prompt: "Explain programming."

  • ✅ Clear Prompt: "Explain the difference between lists and tuples in Python with code examples."

2. Instruction-based Prompting

  • Clearly instructing the model on what to do enhances accuracy.

  • ❌ General Question: "What is HTML?"

  • ✅ Instruction-based Prompt: "Explain the role of HTML and describe five basic tags with examples."

3. Contextual Prompting

  • Including background information in the prompt leads to more relevant responses.

  • ❌ Simple Question: "What is functional programming?"

  • ✅ With Context: "Compare the concept and differences of functional programming in JavaScript and Python."
