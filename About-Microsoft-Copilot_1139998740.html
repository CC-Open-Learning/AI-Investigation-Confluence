AI Investigation : About Microsoft Copilot

Created by Josslyne Shantz (Unlicensed), last modified on May 29, 2024

Microsoft Copilot is Microsoft's AI chatting interface that is integrated into Microsoft 365. It combines Large Language Models (LLMs) - including GPT-4 and GPT-4 Turbo - with Microsoft 365 apps and your data in the Microsoft Graph. Microsoft Graph is an API that connects to multiple services and devices to collect data.

Microsoft Copilot also has a Business Chat feature, allowing you to ask more things from it, such as asking it to tell you how your team has updated something. Copilot will go through your business data (e.g. meeting, chat threads, email) to tell you what you want to know.

Microsoft Copilot can create content to start with and edit based off a prompt, such as telling it to draft a proposal based off a file of notes.

Copilot has 3 plans. The first is the free plan, next is the pro plan which is currently $27 a month, the last is the business plan which is currently $40.70 a month per user.

The free plan allows for document and web page summarization; chatting with text, image, and voice capabilities; image creation in the Designer which was formerly the Bing Image Creator; web grounding; and the use of plugins and Copilot GPTs, along with commercial data protection for free if you are an eligible Microsoft Entra ID user.

The pro plan allows for everything in the free plan along with faster responses during peak times; Copilot capabilities in the web versions of Word, Excel, OneNote, PowerPoint, and Outlook, and faster AI image creation.

The business plan allows for everything in the free plan along with Copilot capabilities in Microsoft 365 Word, Excel, OneNote, PowerPoint and Outlook in English; Copilot capabilities in Microsoft Teams; enterprise grade data protection; Microsoft Graph grounding; and use of Microsoft Copilot Studio.

Multiple Copilots can be created using Microsoft Copilot Studio.

Copilots can engage with users across websites, Facebook, Microsoft Teams, websites or any channel that is supported by the Azure Bot Service.
