AI Investigation : Proposed Tech Stack

Created by Diego Andres Bolanos Osejo (Unlicensed), last modified on Jun 21, 2024

Implementation Outline

1. Backend

  • Setup NestJS with Node.js (Express.js):

    • Create a NestJS server to handle backend logic and API requests.

    • Use Node.js (Express.js) as the framework for server-side operations.

  • Integrate AI Services:

    • Azure OpenAI: Connect to Azure's OpenAI services to leverage powerful language models.

    • Azure Cognitive Service for Language: Utilize Azure's cognitive services for natural language processing tasks.

    • LangChain: Integrate LangChain to handle various AI functionalities such as data retrieval and query processing.

2. Frontend

  • Develop User Interface:

    • Use Angular to create a responsive and dynamic frontend.

    • Implement TailwindCss for styling the user interface components.

3. AI Integration

  • Connect to Brightspace API:

    • Set up API connections to fetch and process educational content from Brightspace.

    • Use LangChain to build and deploy the chatbot, handling data retrieval and response generation.

  • Create Chatbot Logic:

    • Implement the logic to handle user queries, leveraging Azure OpenAI and Azure Cognitive Services for natural language understanding and generation.

    • Use LangChain for managing message history, querying documents, and integrating with external APIs.

4. Deployment

  • Containerization and Orchestration:

    • Use Docker to containerize the application.

    • Deploy using Kubernetes for scalability and manageability.

  • Azure Deployment Services:

    • Use Azure DevOps for continuous integration and continuous deployment (CI/CD) pipelines.

    • Deploy the application on Azure App Service for scalable and reliable hosting.

5. Storage

  • Cloud Storage Solutions:

    • Azure Cosmos DB: Use Azure Cosmos DB for storing chatbot data and large educational content.

    • Azure Database for MySQL: Utilize Azure's MySQL database for relational data storage.

    • Azure Blob Storage: Store large files and unstructured data.

    • Azure File Storage: Use file storage for static files and resources.
