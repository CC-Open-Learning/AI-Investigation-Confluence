1. Role Prompting (Assigning a Role)

  • Assigning a specific role to the model helps generate more professional and domain-specific responses.

2. Negative Prompting (Restricting Unwanted Responses)

  • Instructs the model to avoid using certain words or response styles.

ðŸ”¹ Example:

# Negative prompting example
negative_prompt = """
Explain quantum computing, but do NOT use words like 'maybe' or 'I think'.
"""

print("Negative Prompting Example:")
print(negative_prompt)

ðŸ”¹ What happens?
The model avoids uncertainty and provides a more confident and direct answer.

3. Multi-turn Prompting (Conversational Prompting)

  • Design prompts that maintain context across multiple interactions.

ðŸ”¹ Example:

# Simulating a multi-turn conversation
conversation = [
    ("User", "What is a GPT model?"),
    ("AI", "GPT is a natural language processing model developed by OpenAI. Do you want to know more details?"),
    ("User", "How does the model learn?"),
    ("AI", "GPT is based on the Transformer architecture and learns from large-scale text data to generate natural language responses.")
]

print("Multi-turn Prompting Example:")
for role, message in conversation:
    print(f"{role}: {message}")

ðŸ”¹ What happens?
The model remembers context from previous interactions, making conversations more coherent and relevant.

import openai

response = openai.ChatCompletion.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "Summarize the history of AI."}],
    temperature=0.5,  # Moderate creativity
    max_tokens=200  # Limits response length
)

print("Optimized Response:", response["choices"][0]["message"]["content"])

âœ… Combining with RAG (Retrieval-Augmented Generation)

  • Enhances the prompt by incorporating real-time retrieved information.

ðŸ”¹ Example (Python - Fetching External Data for RAG)

import requests

def search_wikipedia(query):
    url = f"https://en.wikipedia.org/api/rest_v1/page/summary/{query.replace(' ', '_')}"
    response = requests.get(url)
    
    if response.status_code == 200:
        return response.json().get("extract", "No summary available.")
    return "Failed to retrieve information."

# Example: Using retrieved data in a prompt
topic = "Artificial Intelligence"
retrieved_info = search_wikipedia(topic)

prompt = f"""
Using the latest information retrieved from Wikipedia:
{retrieved_info}

Now, summarize this information in a clear and concise way.
"""

print("RAG-based Prompting Example:")
print(prompt)
