AI Investigation : Brightspace Export Data Explanation

Created by Arsh Chauhan, last modified on Jul 30, 2024

We can export components from the course offering to a file. This will allow us to reuse the components in another system including Brightspace or to store the components then re-import them into another course offering a template. Components that are exported to zip file are standard file based on the IMS Content Packaging specification.

1. Mapping Required Data Points to Brightspace Course Components

Required Data Points and Corresponding Brightspace Components:

  • Announcements: Contains important course updates and reminders.

  • Assignments: Includes student submissions and instructor feedback.

  • Calendar Events: Lists key dates for course activities.

  • Checklists: Tracks student progress on various tasks.

  • Competencies: Assesses student skills and learning outcomes.

  • Content: Core teaching materials, including modules and readings.

  • Course Files: All files uploaded as part of the course.

  • Discussions: Online discussions between students and instructors.

  • External Links: Links to external resources and websites.

  • FAQs: Frequently asked questions related to the course.

  • Glossary: Definitions of key terms used in the course.

  • Grades: Student grades on assignments and assessments.

  • Question Library: Repository of questions used in quizzes and exams.

  • Quizzes: Assessments to test student knowledge.

  • Rubrics: Criteria for grading assignments and assessments.

  • Self Assessments: Tools for students to assess their own learning.

  • Surveys: Tools for gathering feedback from students.

Exporting Course Components

We can also exports components of the course as a course package.

To Export course components

  1. In the course offering containing the components you want to copy, on the navbar, click Course Admin â†’ Import/Export/Copy Components.

  2. Select one of the following options there

    1. Export as Brightspace Package

    2. Export as Common Cartridge

    3. Export as Thin Common Cartridge

  3. Click Start.

  4. In the Select Components to Export section, to export all components, select the Select All Components check box or select the check box for the particular components you want to export.

  5. To export only certain items from a component, select the Select individual items to export radio button.

    (If you chose Select individual items to export for any component in the previous step, the Select Items page appears. This page lists all of the items and any sub-items contained in the current component. Otherwise, the Confirm Components to Export page appears. To view sub-items, click the + Expand icon next to an item. Select the individual items and sub-items you want to export and click Next. When you have finished selecting individual items from components, the Confirm Components to Export page appears)

  6. Click Continue.

  7. The Export started dialog displays to confirm the export is underway.

  8. Click Done.

  9. When the export package is ready for download, a notification is generated and can be accessed from the bell icon in the navigation bar.

  10. Click the Export finished notification message. This directs you back to the Import/Export/Copy Components tool in your course.

  11. Click the Click here to download the export Zip package link.
