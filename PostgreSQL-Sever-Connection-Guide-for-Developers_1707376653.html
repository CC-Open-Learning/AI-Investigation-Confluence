Created by Eunie, last modified on Mar 21, 2025

Access to the database is limited to the Conestoga College Wi-Fi network.

If you require access from an external IP address, please submit a request.

PostgreSQL Connection Guide

This guide uses pgAdmin 4 to connect to the PostgreSQL database. However, feel free to use tools like DBeaver, terminal (psql), or any tool you’re comfortable with.

Connecting via pgAdmin 4

1. Installation

  • Download and install pgAdmin from the official pgAdmin website.

2. Connection Steps

  • Launch pgAdmin and either

    • Right-click on “Servers” in the left sidebar → select “Create” → “Server”

    • (or Click “Add New Server” from the Quick Link section.)


  • Under the [General] tab

    • Name: Enter a display name (e.g., CVRI-AI-Server).



  • Switch to the [Connection] tab and enter the following:

Host name/address: cnerg-data.canadacentral.cloudapp.azure.com

Port: 8086

Maintenance database: cvri-course-db

Username: aiadmin

Password: (The password will be shared separately.)

  • Click [Save] to complete the connection.
