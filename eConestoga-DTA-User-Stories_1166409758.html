AI Investigation : eConestoga DTA User Stories

Created by Josslyne Shantz (Unlicensed), last modified on Jun 19, 2024

User Story: As a student, I want to be able to find when my exam is happening easily so that I can plan my study schedule.

Acceptance Criteria: Given that I ask when my exam is, when I click on the send button, then I should receive a response with my exams time.

User Story: As a student, I want to find when/where a topic was talked about so that I don’t have to look through all the content available and potentially miss what I need because I forgot when/where it was talked about.

Acceptance Criteria: Given I ask about a topic, when I click the send button, then I should receive a response with some information and a link to where the information is.

User Story: As a student, I want to be able to ask some clarifying questions so that I don’t have to send an email to the teacher and wait to get a reply.

Acceptance Criteria: Given I ask a simple clarifying question, when I click the send button, then I should receive a response answering the question and giving internal sources to study further.
