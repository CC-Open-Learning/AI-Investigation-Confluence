  • Why to use Copilot Studio:

    • To create personalization copilots, we can feed the organization’s data and get narrowed/personalized output.

  • Can be deployed in live website: https://learn.microsoft.com/en-us/microsoft-copilot-studio/publication-connect-bot-to-web-channels?tabs=preview

=> Workflow to create and deploy chatbot:

  1. For external questions about the Conestoga website, we will need only the website’s link.

    1. Aerish has already made that copilot. We can fine-tune that.

  2. For internal questions(on Conestoga), we can generate prompts(inputs for API) and get answers using bright space Apis.

    1. https://learn.microsoft.com/en-us/microsoft-copilot-studio/authoring-http-node

  3. We can then deploy our copilot on a live website.

    1. https://learn.microsoft.com/en-us/microsoft-copilot-studio/publication-connect-bot-to-web-channels?tabs=preview
