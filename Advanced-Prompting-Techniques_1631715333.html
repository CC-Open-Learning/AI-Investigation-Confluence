2. Chain-of-Thought (CoT) Prompting (Step-by-step Thinking)

This technique encourages the model to think logically and break down problems step by step.

ðŸ”¹ Example:

# Problem
Q: A farmer has 5 chickens, 3 cows, and 2 sheep. How many legs do they have in total?  

A:  
1. Chickens have 2 legs each. 5 Ã— 2 = 10  
2. Cows have 4 legs each. 3 Ã— 4 = 12  
3. Sheep have 4 legs each. 2 Ã— 4 = 8  
4. Total: 10 + 12 + 8 = 30  

Therefore, the total number of legs is **30**.

ðŸ”¹ What happens?
Instead of guessing, the model solves the problem step by step, leading to more accurate answers.

3. Self-Consistency Prompting (Choosing the Most Common Answer)

This method generates multiple answers and selects the most frequent one to ensure consistency.

ðŸ”¹ Example:

import random
from collections import Counter

# Simulating multiple AI responses
def generate_answer():
    answers = ["Paris", "London", "Berlin", "Paris", "Paris", "Berlin", "Paris"]
    return random.choice(answers)

# Get multiple answers
responses = [generate_answer() for _ in range(10)]

# Count occurrences
most_common_answer = Counter(responses).most_common(1)[0][0]

print("Generated rã…‹esponses:", responses)
print("Most consistent answer:", most_common_answer)

ðŸ”¹ What happens?
The model may give different answers each time. To get the most reliable answer, we run multiple responses and choose the most frequent one.

4. ReAct (Reasoning + Acting)

This technique combines thinking (reasoning) and doing (acting). It is useful for tasks that require external information, such as searching the web.

ðŸ”¹ Example:

import requests

def search_wikipedia(query):
    url = f"https://en.wikipedia.org/api/rest_v1/page/summary/{query.replace(' ', '_')}"
    response = requests.get(url)
    
    if response.status_code == 200:
        return response.json().get("extract", "No summary available.")
    return "Failed to retrieve information."

def react_example(query):
    # Step 1: Thinking
    print(f"Thinking: I need to find information about '{query}'.")

    # Step 2: Acting
    print("Acting: Searching Wikipedia...")
    result = search_wikipedia(query)

    # Step 3: Generating Answer
    print("Final Answer:")
    print(result)

# Example usage
react_example("Artificial Intelligence")

ðŸ”¹ What happens?

  • First, the AI decides what to do (Reasoning).

  • Then, it acts (searching Wikipedia).

  • Finally, it provides an answer based on real data.
