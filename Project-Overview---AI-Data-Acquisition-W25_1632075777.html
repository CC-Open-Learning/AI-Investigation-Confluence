Background

Create a Digital Teaching Assistant (DTA), powered by Generative AI (GenAI) to support learner success. It will be able to:​

  • Summarize materials, answer questions, and provide personalized guidance​

  • Deliver dynamic case studies as a summative activity for a Course Unit, through an interactive GenAI chat experience.​

The team built a Proof of Concept with a PMP/ITIL course last term, providing accurate summaries and reliable Q&A to help students review and understand class content effectively​

  • We have validated how to build this of tool​

  • This PoC is very limited and not suitable for deployment

Value

Case Study Simulation​

  • Deliver interactive chat experiences for learners to repeat, rehearse, and master Unit Level Outcomes​

  • Infinite re-playability​

  • Hypothesis: These will be able to be created by faculty with no-code, via GenAI Chat​

Course Import Process:​

  • Develop a high-quality, sustainable process to import course content ​

  • Production deployment will require all participating courses to be imported each semester there are content changes

Purpose

The goal is to iterate on the Digital Teaching Assistant (DTA) to support at least one course (up to five) from the ACSIT program, improving student learning with accurate summaries and Q&A​

  • This will result in a prototype that significantly expands the robustness and scope of the proof of concept​

Our initial target for candidate courses are from 2nd semester SET and/or CPA​

  • Our dev team are experts in this material which will significantly accelerate development

Scope

In-scope

Data Formats: ​

  • Ideal course maximizes course content available, especially a complete set of lecture recordings​

  • Only course content is in-scope​

  • Course outlines, Learning outcomes, Lecture slides (Ppts, Pdfs), Web linkS, Videos, Spreadsheets, Documents, Transcriptions of lectures, H5P content OLC content, etc. ​

​

Course Structure & Scheduling: ​

  • Provides a clear sequence of units, including dates, schedules, and topic order​

  • Centralized access to quiz dates, study topics, and assignment deadlines​

  • Reduces student inquiries and improves efficiency for both instructors and students​

​

Accessibility Check: ​

  • Review and confirm accessibility compliance for all data formats​

Out of-scope

  • Course submissions: all student work, contributions, etc. are fully excluded from this project​

  • Chatbot features exclude grading/assessment​

  • Instructors are not required as stakeholders
