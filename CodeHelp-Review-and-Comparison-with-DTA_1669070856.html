Paper Summary: CodeHelp – LLM-Based Programming Learning Support Tool

Background

  • In large programming classes, providing individual support to students is challenging.

  • LLM-based AI tools can offer instant assistance, but students may over-rely on direct answers.

  • CodeHelp is an LLM-powered tool designed to guide students in problem-solving without directly providing answers.

- Paper: CodeHelp: Using Large Language Models with Guardrails for Scalable Support in Programming Classes

Key Features of CodeHelp

1. Student Support Features

  • Analyzes error messages and code to offer guidance without giving direct answers.

  • Requests additional details if the question lacks sufficient information.

  • Instructors can restrict specific keywords (e.g., sum() function).

2. LLM-Based Response Process (3 Steps)

  • Query Review: Checks if the provided information is sufficient.

  • Response Generation: Provides hints and explanations without revealing complete solutions.

  • Code Removal or Transformation: If code is included, it is replaced with a descriptive explanation.

3. Instructor Support Features

  • Enables instructors to monitor student queries and AI responses.

  • Provides options to restrict advanced functions to align with course content.

  • Supports LMS integration (Moodle, Canvas, etc.).

Evaluation and Findings

  • Deployed in a first-year computer science course (52 students, 12 weeks) at Illinois Wesleyan University.

    • Over 70% of students used CodeHelp regularly, with some submitting over 100 queries.

    • More than 80% found it helpful for assignments, and 95% wanted to use it in other courses.

    • Instructors found it useful for tracking student understanding and providing feedback.

Limitations and Future Improvements

  • LLM responses can occasionally contain incorrect information.

  • Students may struggle to phrase effective queries.

  • Future research should focus on personalized feedback and evaluating the impact of learning.

Conclusion

  • CodeHelp is an LLM-powered educational tool that enhances programming learning by guiding students rather than providing direct answers.

  • It serves as a valuable complement to traditional instruction for students and educators.
