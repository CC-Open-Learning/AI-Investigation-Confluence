AI Investigation : ETL Process and Dataset Preparation for prototype

Created by Aerish Gaba (Unlicensed), last modified on Jul 10, 2024

Overview

This document outlines the steps taken to extract data from PDF and video files, perform ETL (Extract, Transform, Load) operations, and prepare the dataset for fine-tuning the Gemma-2 9B parameter model.

Data Extraction

Extraction from PDF Files

  1. Tools Used:

    • PyMuPDF for PDF processing.

    • pdfplumber for extracting text from PDFs.

  2. Steps:

    • Read PDF Files: Load the PDF files using PyMuPDF or pdfplumber.

    • Extract Text: Extract text content from each page of the PDF.

    • Store Data: Save the extracted text to a structured format such as JSON or text file.

Extraction from Video Files

  1. Tools Used:

  2. Steps:

    • Extract Audio: Use to extract audio from video files.

    • Convert Speech to Text: Use to transcribe audio to text.

    • Store Data: Save the transcriptions in a structured format such as JSON or text files.

ETL Process

Extract

  1. Source Identification:

    • Identify all PDF and video files containing relevant course content.

    • Consolidate all files into a single directory for processing.

Transform

  1. Cleaning Data:

    • Remove unwanted characters and formatting issues from extracted text.

    • Normalize the text (e.g., converting to lowercase, removing stopwords).

  2. Structuring Data:

    • Organize extracted content into a structured format (e.g., JSON) with appropriate metadata/LLM from transformers (e.g., module names, topics).

Load

  1. Storing Data:

    • Save the cleaned and structured data to a persistent storage solution (e.g., a database, cloud storage, or local files).

Preparing Dataset for Fine-Tuning

  1. Dataset Formatting:

    • Ensure the structured data is formatted correctly for the fine-tuning process.

    • Typically, the data needs to be in a JSONL format, where each line is a JSON object representing a single training example.
